# Банковская база данных 

## Описание

Реализация реляционной базы данных для банковской системы с тестовыми данными и набором аналитических SQL-запросов. 

## Схема базы данных

- `clients` - клиенты банка
- `accounts` - счета (дебетовые, сберегательные, кредитные)
- `cards` - банковские карты
- `transactions` - транзакции
- `loans` - кредиты
- `loan_payments` - платежи по кредитам

## Запуск базы данных

1. Установите Docker и Docker Compose
2. Выполните в корне проекта:

```bash
docker-compose up -d
```

База будет доступна на localhost:5432

Параметры подключения:
- БД: `bank_db`
- Пользователь: `admin`
- Пароль: `admin123`

## Выполнение SQL запросов

### Подключение к базе:

```bash
docker-compose exec postgres psql -U admin -d bank_db
```

### Выполнение конкретного запроса из файла:

```bash
docker-compose exec postgres psql -U admin -d bank_db -f /queries/query_01.sql
```

### Выполнение всех запросов последовательно:

```bash
for i in {1..10}; do
  docker-compose exec postgres psql -U admin -d bank_db -f /queries/query_${i}.sql
done
```

## Описание запросов

1. **query_01.sql** - Найти активные счета в EUR, открытые после 2024-01-01
2. **query_02.sql** - Вывести ФИО клиента, тип счета, валюту и статус счета
3. **query_03.sql** - Вывести всех клиентов и количество их счетов (включая 0)
4. **query_04.sql** - Найти клиентов, у которых больше 2 активных счетов
5. **query_05.sql** - Найти счета, у которых сумма входящих операций выше среднего по банку
6. **query_06.sql** - Топ-5 клиентов по сумме всех операций за 2025 год
7. **query_07.sql** - Определить активность клиента по количеству операций за последние 90 дней
8. **query_08.sql** - Найти кредиты, по которым сумма успешных платежей < 50% от principal
9. **query_09.sql** - Показать все активные карты и кому они принадлежат
10. **query_10.sql** - Для каждого счета посчитать количество операций и сумму списаний

## Управление контейнерами

Остановить контейнер:

```bash
docker-compose down
```

Остановить и удалить данные:

```bash
docker-compose down -v
```

Просмотр логов:

```bash
docker-compose logs -f postgres
```
