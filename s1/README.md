# Система управления спутниковой группировкой | Семинар 1

Проект представляет собой объектно-ориентированную систему для управления группировкой искусственных спутников Земли.

## Компиляция и запуск

Запуск производится из корня проекта, директории s1

### 1. Через Gradle (рекомендуется)

```bash
.\gradlew build
.\gradlew run

```

#### Тесты

.\gradlew test

### 2. Через Java (ручная компиляция)

```bash
javac -d app/build/classes/java/main app/src/main/java/ru/bmstu/sd/s1/App.java app/src/main/java/ru/bmstu/sd/s1/satellite/*.java

java -cp "app/build/classes/java/main" ru.bmstu.sd.s1.App
```

## О проекте

Система моделирует управление различными типами спутников:
- Спутники связи (CommunicationSatellite) для передачи данных
- Спутники дистанционного зондирования (ImagingSatellite) для съемки территории
- Управление группировками спутников (SatelliteConstellation)

### Основные функции:

- Активация/деактивация спутников
- Выполнение специализированных миссий
- Управление энергопотреблением
- Автоматический контроль уровня заряда батареи

### Технические требования:

- Java 11 или выше
- Для сборки через Gradle: JDK с поддержкой Java 17

### Особенности реализации:

- Абстрактный класс Satellite как базовый для всех типов спутников
- Полиморфное выполнение задач через общий интерфейс
- Полное покрытие модульными и интеграционными тестами
- Корректная обработка граничных условий и ошибок
